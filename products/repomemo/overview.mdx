RepoMemo helps development teams capture and maintain <Tooltip tip="Technical documentation that explains code structure, APIs, and usage">repo documentation</Tooltip> alongside your codebase. Scan repositories, generate docs from <Tooltip tip="Source code comments formatted for automatic documentation extraction">docstrings</Tooltip>, and keep knowledge in sync with your code.

<Badge color="gray">Planned</Badge>
