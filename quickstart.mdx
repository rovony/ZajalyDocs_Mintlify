## Prerequisites

<Info>
  **You'll need:** Node.js v20.17.0+ (LTS recommended) and Git installed on your machine.
</Info>

## Set up your environment

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone git@github.com:rovony/ZajalyDocs_Mintlify.git
    cd ZajalyDocs_Mintlify
    ```
  </Step>
  <Step title="Install the Mintlify CLI">
    ```bash
    npm i -g mint
    ```
  </Step>
  <Step title="Start the local preview">
    ```bash
    mint dev
    ```

    Open [http://localhost:3000](http://localhost:3000) to see the docs. Changes auto-reload as you edit files.

    <Check>You should see the Zajaly Developer Docs home page at localhost:3000.</Check>
  </Step>
</Steps>

## Make your first edit

<Steps>
  <Step title="Open any .mdx file">
    Every page is an MDX file (Markdown + JSX components) with YAML frontmatter at the top:

    ```yaml
    ---
    title: "Page title"
    description: "Brief description for SEO."
    ---
    ```
  </Step>
  <Step title="Edit and preview">
    Save the file. Your browser refreshes automatically at localhost:3000.
  </Step>
  <Step title="Deploy">
    Push to the `main` branch. Mintlify auto-deploys â€” no build commands needed.

    ```bash
    git add .
    git commit -m "Update docs"
    git push origin main
    ```

    <Check>Your changes are live at [docs.zajaly.dev](https://docs.zajaly.dev) within minutes.</Check>
  </Step>
</Steps>
